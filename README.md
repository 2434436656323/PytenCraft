# Как работать с Pyten Craft:

* Во‑первых, игра написана на Python.
* Была скомпилирована.
* Всё было проверено админами.
* Работает только на Windows (даже если на Linux или macOS подрубить Wine).

# Управление:

* Игрок управляется стрелками, но можно и не управлять.
* 1–7 — это блоки (пока есть):
  * 1 — земля;
  * 2 — камень;
  * 3 — уголь;
  * 4 — трава;
  * 5 — медь;
  * 6 — железо;
  * 7 — алмазы.
* Выживания нет, Full креатив.
* На F5 мир сохраняется, а на F6 вставляется.
